Movie Poster Design

Tool(s): Adobe Illustrator

Project Description: I used illustrator to create simple shapes and organized those shapes to create this poster. I used a gradient effect on a shape to create the background. I also used a radial gradient to create to glow from the computer. I used the pen tool to create a curved path for the text coming from the computer.
